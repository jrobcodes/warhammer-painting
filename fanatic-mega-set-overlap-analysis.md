# Fanatic Mega Set vs. My Collection — Overlap Analysis

## Summary
The Fanatic Mega Set (~50 paints, ~$140) has massive overlap with my existing ~210 Citadel collection. Not recommended as a bulk purchase.

## Fanatic Mega Set Contents

### Colors (36)
Matt Black, Deep Grey, Ash Grey, Matt White, Thunderous Blue, Wolf Grey, Ultramarine Blue, Imperial Navy, Crystal Blue, Bright Sapphire, Abyssal Blue, Hydra Turquoise, Marine Mist, Pharaoh Guard, Aqua Alchemy, Angel Green, Greenskin, Rainforest, Autumn Sage, Army Green, Necrotic Flesh, Desert Yellow, Daemonic Yellow, Ice Yellow, Ancient Stone, Oak Brown, Fu Brown, Pure Red, Lava Orange, Wicked Pink, Alien Purple, Magecast Magenta, Mulled Berry, Basilisk Red, Wyvern Fury

### Skin Tones (4)
Dorado Skin, Barbarian Flesh, Topaz Skin, Onyx Skin

### Metallics (3)
Weapon Bronze, Greedy Gold, Plate Mail Metal

### Effects (3)
True Blood, Fresh Rust, Oozing Vomit

### Washes (4)
Soft Tone, Strong Skin Shade, Dark Tone, Strong Tone

## Key Overlap Areas
- Black/White/Grey: Already have Abaddon Black (x4), Matt White, Mechanicus Standard Grey, etc.
- Blues: Macragge Blue (x2), Caledor Sky, Hoeth Blue, Lothern Blue
- Reds: Mephiston Red, Evil Sunz Scarlet, Wild Rider Red
- Greens: Warpstone Glow, Waaagh! Flesh, Caliban Green
- Browns: Mournfang Brown, Rhinox Hide, Dryad Bark
- Metallics: Leadbelcher, Retributor Armour, Ironbreaker, Balthasar Gold
- Washes: Nuln Oil (x3), Agrax Earthshade (x6!), Seraphim Sepia (x2)

## Recommendation
Buy individual Fanatic paints as replacements when Citadel pots run out, rather than the Mega Set. Consider the Complete Set ($800-1000) only if going all-in on dropper bottles.
